1、isStatic：检测数据是不是除了symbol之外的原始数据

2、isPrimitive:监测数据是不是原始数据

3、isObject:判断数据是不是引用类型的数据（例如：array,function,object,regexe,new Number(),new String())

4、isObjectLike: 判断value是否是类对象，如果一个值是类对象，那么他不应该是null，而且typeof之后的结果是“object”

5、getRawType:获取数据类型，返回结果为Number、String、Object、Array等

6、isPlanObject:判断数据是不是Objecy类型的数据

7、isArray:判断数据是不是数组类型的数据（Array.isArray的兼容写法）

9、isRegExp：判断数据是不是正则对象

10、isDate:判断数据是不是时间对象

11、isNative:判断value是不是浏览器内置函数

13、isFunction: 检查value是不是函数

14、isLength：检查value是否为有效的类数组长度

15、isArrayLike：检查value是否是类数组

16、isEmpty：检查value是否为空

17、cached：记忆函数：缓存函数的计算结果

18、isPCBroswer：检测是否为PC端浏览器模式

19、dateFormater：格式化时间

20、getUrlParam:获取url参数，返回一个对象

23、exitFullScreen: 退出全屏
