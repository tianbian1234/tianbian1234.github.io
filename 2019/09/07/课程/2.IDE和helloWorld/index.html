IDE和helloWorld
教学
Publish Date:   2019-09-07
Read Count:  

今天开始我们学习C语言的第一节课，在学习C语言之前的，我们要先准备一下学习的工具，
工欲善其事，必先利其器。我们先来准备一下我们的写代码工具，IDE，是不是有很多人，不理解这个是什么
东西？IDE就是编程语言编辑器，就像我们写作文，需要作文本。我们学数学，需要计算器一样？要想学代码，那么
没有一个写代码的工具又怎么学呢？
IDE有很多种，dev c++，Visual Studio，Visual C++ 6.0，sublime Text等等，这些都是比较常见
编程语言编辑器，我给大家选择的是dev c++，为什么选择这个呢？我们先安装，然后做一下小实验
你们、就可以体会到了，安装包我放在了桌面，大家可以打开就能使用了。双击之后一路next就可以安装成功了。

IDE的选择（初学者的困惑）

1.用什么软件比较好，选择一个适合自己并且适合机器的东西
2.当代语言往往只有一个可用的软件。
3.一起使用keil吧！为了未来更好的结合嵌入式的单片机开发
4.keil安装链接

IDE 是 Intergreated Development Environment 的缩写，中文称为集成开发环境，用来表示辅助程序员开发的应用软件，是它们的一个总称。

为什么要选择IDE呢？

1.使用记事本的时代
使用记事本写一个hello，word
2.使用IDE的好处
使用IDE写一个hello world

两个做比较实验一下
记事本，
1.读写不方便
2.容易出现语法错误
3.没有补全的效果（这个是好处也是坏处）

IDE
1.读写操作很方便
2.不容易出现语法的错误，一但出现错误，会有错误提醒标识
3.拥有补全的效果。
4.结构代码清晰，可以分颜色展示，容易理解词汇的用途

第一个c程序

程序的框架

头文件 #include <stdio.h>
1.以#开头语句称为预处理指令（在程序开始之前就已经处理好的）
2.#include在程序中并不是必须的，但是一旦存在，那么必须放在程序的开始处

stdio实际上使 standared input output header的缩写
标准 输入 输出
h是header的缩写，也就是头的意思，这里标识头文件。

整体的翻译也就是“预处理包含->标准的输入输出头文件”

输入其实就是我们通过程序赋予程序的东西
输出呢其实就是程序的输出结果，我们看到的东西

例如对于这个电脑来讲，我们用键盘打字，你把c输入到电脑里面了。这就叫输入
这时候呢？你看到屏幕上出现了一个C，这就叫输出，屏幕输出了C。

也就是说，一道数学题，比如1 + 1 = ？
这里1 + 1就叫做输入
2是所得的结果，这个就叫做输出。

主函数
int main(){
return 0;
}

1.main函数是程序的起点/入口。
其中的return是返回的意思.也就是程序的输出
main函数可以返回一个值，也可以没有返回值。如果这个函数没有返回值，那么前面的int就可以替换为void！

尽量不要用void的形式，这个方式有的编译器不支持，有可能出错，尽量用int return 0的方式就好了。

2.printf(‘hello, world’)的作用是在屏幕上打印一行文字：”hello world！”并且换行“\n”
最后的分号表示语句结束，注意不要使用中文符号。

注意：C程序中的一句话可以跨行运行，并且用分号通知编译器该语句已结束。

注意在 int main{
这里是我们之后的重点位置
return 0
}
1.课程所有目前需要的东西都是有着一段作为模版生成的。
2.直到我们学习函数之前，我们代码执行片段都只需要在这中间展示

我们以后所有的东西都是在这中间去写的。

无规矩不成方圆
1.大括号必须有并且要单起一行
2.缩紧的你要行
3.注释，为什么要添加注释呢？
单行注释 //
多行注释

**/

编译和执行

这里只做简单的介绍

编译

把我们的源文件，经过初级翻译变成目标代码/文件，目标代码是编译器的输出结果，它拓展名一般为’.o‘或者’.obj’

连接

将目标代码和c函数库连接，并将源程序中的目标代码和库代码合并，形成最终的可执行代码
注意：最终可执行程序代码指的是经过编译程序最终翻译成的机器语言，也就是机器可以按照步骤执行你的指令了。

执行

在特定的机器环境下执行可执行程序
注意：为什么要是特定的环境呢？这个就是房子一样，大的适合大人住，小屋子适合小孩住，写程序的目的就是
解决某种环境下的某个问题，比如你现在要计算自己的体重值多少钱，那肯定是按照现在的环境去写程序的，
你说等明年了，你再拿着现在的程序去计算，是不是就不恰当了，所以强调了一下特定机器环境。这里以后我们
会逐步深入讲解，现在你只需要有所了解即可，到了单片机的阶段，就会更清晰，更明白。
