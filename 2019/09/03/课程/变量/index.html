变量与常量数据解析

在程序的指导下，程序可以做很多的事情，如数值计算，名字排序，执行语言或视频命令，计算彗星轨道，准备邮件列表，拨打电话号码，画画，做决策或者
是你能想到的其他事情。想要完成这些任务，程序需要使用数据，即承载信息的数字或者字符。有些数据类型在程序使用之前就已经预设好了，在程序的
整个运行过程中都不会变化，这些称为常量（constant）。其他的数据类型在程序的运行过程中可能会改变或者赋值，这些称为变量（variable）。
比如3.1414926，这就是一个常量，比如weight，这就是一个变量。在比如1700，这个是一个变量还是常量呢？白金的价格在实际生活中是一只变得，
但是在这里它只是一个确定的值，所以这也是一个常量！

常量就像是你买好的房子，只属于你自己，能住的就是你
变量就像是租来的房子，房子虽然不会变，但是当你一段时间打开房子的时候就会发现，总是有不同的人住进去了！

在上面的例子里面我们定义了
const name = “richa”
int num = 10
这里的房子也就是变量就分别叫做name和num，他们的值呢？其实就分别是richa和10.
这个就叫做变量的定义。
也就是在计算机中开辟一块特定的内存空间，这个就是定义课一个变量，那如果定义十个空间怎么办呢？我们也分不清
十个空间都是什么，因为他们都是一样的，对的，就是给他们一个名字，每个不同的名字标志一个空间，这样我们就可以
找到自己的家了对吧？

赋值也就是让这个空间有了住户，就是把数据放进到空间中，比如 num = 10，这句话就是把10放入num的空间里

变量命名规则

给变量命名时要注意使用有意义的变量名和标识符（例如程序中需要使用变量来计算年龄，这个时候你就要使用age，如果在具体一点，说
小狗的年龄，就是dog_age，而不是使用x1什么鬼的东西来表示）这里实际上有一个规则，在变量名中只能出现数字，字母，下划线等，并且
数字不能位于开头的位置，例如 3dog，这个就是一个错误的变量，我们可以实验一下。

变量名是区分大小写的，有的人会问，怎么区分？其实是这样的，Dog_age和dog_age就是两个变量名了。

变量名最好使用具有语义话的变量名，例如dog_age

1.c语言规定变量名只能由字母、数字、下划线3种字符组成。
2.变量名的首字母只能是字母或者下划线，不能是数字类型。
3.变量名的其他字母可以是下划线，数字和字母。
4.不能使用关键字（保留字）一共32个，只是稍微统计，现在更多增加了。

正确的变量名

dog_age, _age_dog, dog_age202

错误的变量名

3dog_age 6_pig Cat!

还有必须记得，编写程序的时候尽量不要用中文，要是说不会英文，容易，找一个你明白的翻译，也比中文靠谱。
你打中文字的时候太容易引起程序错误了。

这就和咱们古代封建社会时期起名字一样，那会皇帝的名字，别人是都不能带着字的，比如说爱新觉罗·玄烨。
他是皇帝，你总也不能叫玄烨吧，会被来出去杀头的。这个关键字就相当于皇帝的名字，你在程序里面是不能
用的。

这就相当于注册公司，马云的公司叫阿里巴巴，你说我的也叫阿里巴巴，完了，人家告了你了，说你侵权，你又得
赔钱，还得改名，倒霉吧，所以变量的关键字就相当于以经注册了的商标，你不能用了，别的你随便，凡是C语言
大神注册了的，你就不能用了。

这里应当插入一下数据类型的解说。

C语言的数据类型 -> 数值型和非数值型
数值型 -> 整型和浮点型
整型 -> int，long，short
浮点型 -> float， double

非数值型 -> char

这些的取值范围可以忽略，我们主要的记住两种类型，int和double，其他的类型使用的不是很多，有兴趣的
人可以自己去研究一下
int类型就是我们常用到的整数，包括正整数，0，负整数。
double类型就是我们通常所说的小数，可以计算到小数点后

生命和使用变量

声明变量

DataType varibaleName               数据类型  变量名

定义时初始化变量

DataType varibaleName = value       数据类型  变量名 = 值

定义后初始化变量

DataType varibaleName               数据类型  变量名
varibaleName = value                变量名 = 值
